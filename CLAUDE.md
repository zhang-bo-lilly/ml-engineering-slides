# CLAUDE.md

This repo holds AIOS team meeting slide decks. Each presentation is in its own `MM-DD-YYYY/` directory.

## Creating presentations

Use the `/aios-slides` slash command. Work one presentation at a time, one slide at a time.

## Directory conventions

- Source files to commit: `*.py`, `*.md`, `requirements.txt`, static assets (`*.pdf`, hand-drawn PNGs not generated by scripts)
- Generated files (never commit): `*.png` produced by scripts, `*.pptx`, `*.docx`, `.venv/`, `__pycache__/`

## Python environment

Each presentation directory has its own `.venv`. Always use `.venv/bin/python` to run scripts within that directory.

## Cleaning

Run `./clean.sh` from repo root to delete all generated artifacts. Use `./clean.sh --venv` to also remove virtual environments.
